# Macarena Davila

Web one-page estática para presentar el servicio de acompañamiento integral para proyectos.

## Estructura

- `index.html`: contenido principal y SEO básico.
- `styles.css`: diseño responsive y sistema visual.
- `script.js`: animaciones suaves al hacer scroll.
- `vercel.json`: configuración simple para publicar en Vercel.

## Publicación en Vercel

Subir esta carpeta como proyecto estático. No requiere instalación de dependencias ni proceso de build.
Actualización para despliegue.
